// turbo-all

Este workflow permite realizar las siguientes tareas de manera directa:

1. Levantar el entorno local: `npm run dev`
2. Ejecutar comandos de base de datos (Supabase CLI)
3. Configurar variables de entorno y archivos de configuración
4. Realizar limpiezas de caché y builds
5. Gestión de archivos del proyecto

Para usarlo, simplemente dime "ejecuta el workflow de desarrollo" para cualquier tarea compleja.
